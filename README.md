# キャラ画像からホームページを自動生成！

このアプリは、キャラクターの画像をアップロードすると：

✅ 背景色を自動抽出  
✅ ChatGPTでキャラ紹介文を自動生成  
✅ おしゃれな紹介ページを表示  

## 🚀 使い方

1. このリポジトリを Render にデプロイ（無料）  
2. アップロードフォームで画像を選ぶ  
3. [ホームページを作成する] をクリック！  
4. 自動生成された紹介ページが表示されます🎉

## 🛠 必要ライブラリ

- Flask
- openai
- colorthief
- gunicorn

## 🌐 Render用設定

- `start.sh` を使って起動します
- `requirements.txt` に必要なパッケージを記載しています
